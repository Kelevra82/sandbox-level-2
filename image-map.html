Image Map

Image maps create clickable regions in a picture.
Hover the mouse over some shapes.

  1. use map and area to define an image map
    • map needs the name attribute
    • img needs the usemap attribute
    • usemap should start with # and match the name of the map
  2. area needs the shape attribute
    • shape can be rect, circle, or poly
    • area needs the coords attribute
    • coords can be obtained with Luna Paint or Pix Spy
  3. add href to area attribute to make it clickable
  4. add title to area attribute for tooltip popups
  5. add target="_blank" to area so links open in a new tab
  6. add link to image map resizer JSDelivr CDN
    • call imageMapResize() to automatically resize image maps
