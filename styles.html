                Two values resizes the width and height separately.

Repeating a background

             
            1.  background-repeat: repeat;
                The background repeats to fill up all the space,
                which may clip (or cut off) the image.
            

             
            2.  background-repeat: no-repeat;
                By default, the background repeats.
            

             
            3.  Repeating a background
                background-repeat: space;
                Repeats the background such that it doesn't get cut off
            

             
            4.  Repeating a background horizontally
                background-repeat: space no-repeat;
                background-repeat: repeat no-repeat;
                The first value is the horizontal repeat (x)
                The second value is the vertical repeat (y)
                Values are seperated by a space
            

             
            5.  Repeating a background vertically
                background-repeat: no-repeat repeat;
                background-repeat: no-repeat space;
                The first value is the horizontal repeat (x)
                The second value is the vertical repeat (y)
                Values are seperated by a space

Background positions

 
            1.  background-position: center;
                The choices are top, bottom, left, right, center
            

 
            2.  background-position: 10%;
                One value specifies both x and y
            

 
            3.  background-position: 20% 30%;
                Two values position x and y individually

Multiple colors gradient

 
                    background-image: linear-gradient(white, gray, green);
                    Add more colors by seperating them with a comma.

Gradient Directions

 
                    background-image: linear-gradient(to bottom right, white, gray, green);
                    to change the direction, add to the beginning:
                    to right

                    to left

Gradient Degrees

 
                    background-image: linear-gradient(90deg, white, gray, green);
                    To change the direction, add degrees to the beginning:
                    90deg (right)
                    135deg (bottom right)
                    270deg (left)
                    225deg (bottom left)

Gradient color stops

 
                    background-image: linear-gradient(90deg, gray, green 10% 90%, gray);
                    Two values next to the middle color sets solid color positions
                    Seperated by spaces

RGBA

 
                color: rgba(6, 68, 0, 0.259);
                Sets the transparency of colors
                Red Green Blue Alpha
                0 alpha is invisible
                1 alpha is fully visible

Opacity

 
                opacity: 0.25;
                Sets the transparency of the element
                0 is invisible
                1 is fully visible

display: none vs opacity: 0

                
                display: none 
                    - makes the element invisible
                    - removes the element from the the browser
                    - the element is still in the element tree

                opacity: 0
                    - makes the element invisible
                    - doesn't remove the element from the browser
                    - the element still takes up space in the browser

Specificity

 
                Styling priority
                1. inline
                2. id
                3. class
                4. element

                Styles are applied based on their specificity or priority.
                Styles with lower priority get overwritten.
                !important overwrites them all.
                !important also goes by load order.
                The !important that gets loaded last gets applied.

Padding shorthand property

 
                padding: 5px 6px 7px 8px;
                Padding gets applied in a clockwise direction when 4 values are specified.

                padding: 10px 20px
                Providing 2 values specifies
                1. top and bottom
                2. left and right

                padding: 10px 20px 30px
                Providing 3 values specifies
                1. top
                2. left and right
                3. bottom

Transformations

 
            1.  transform: rotate(20deg);
            

 
            2.  transform: translate(10px, -10px);

Transitions (animations)

 
                transition: transform 2s;
                The first value is the style to animate
                The second value is the duration of the animation

                Set an animation for the original state to prevent snap-back
